Actix binding for Deno.

## Usage

```ts
import { run } from "https://deno.land/x/actix/mod.ts";

actix();
```

run example:

```bash
deno run -A --unstable https://deno.land/x/actix/example.ts
```

## Flags

- `--unstable`
- `--allow-plugin`
